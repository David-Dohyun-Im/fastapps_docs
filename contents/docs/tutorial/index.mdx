## Tutorial 1: Hello World

### Goal
Create a simple greeting widget that displays a personalized message.

### Python Tool

Create `server/tools/greeting_tool.py`:

```python
from fastapps import BaseWidget, Field, ConfigDict
from pydantic import BaseModel
from typing import Dict, Any

class GreetingInput(BaseModel):
    model_config = ConfigDict(populate_by_name=True)
    name: str = Field(..., description="Person to greet")

class GreetingWidget(BaseWidget):
    identifier = "greeting"
    input_model = GreetingInput

    def execute(self, inputs: GreetingInput) -> Dict[str, Any]:
        return {"message": f"Hello {inputs.name}!"}
```

### React Widget

Create `widgets/greeting/index.jsx`:

```jsx
import React from "react";
import { useWidgetProps } from "fastapps";

export default function GreetingWidget() {
  const props = useWidgetProps();
  return (
    <div style={{ padding: "1rem" }}>
      <h2>Hello Widget</h2>
      <p>{props.message}</p>
    </div>
  );
}
```

### Run It

```bash
fastapps dev
```

- Auto-discovers the widget
- Hot reloads changes
- Launches the widget in ChatGPT preview

### State Updates

Add a search box with widget state:

```jsx
import { useWidgetProps, useWidgetState } from "fastapps";

export default function PizzaList() {
  const { city, pizzas } = useWidgetProps();
  const [query, setQuery] = useWidgetState("query", "");

  const filtered = pizzas.filter((pizza) =>
    pizza.name.toLowerCase().includes(query.toLowerCase())
  );

  return (
    <div style={{ padding: "1rem" }}>
      <input
        value={query}
        onChange={(event) => setQuery(event.target.value)}
        placeholder="Filter pizzas"
      />
      <ul>
        {filtered.map((pizza) => (
          <li key={pizza.name}>{pizza.name}</li>
        ))}
      </ul>
    </div>
  );
}
```
